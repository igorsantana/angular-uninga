O que é AngularJS?

Framework mantido pela Google que é utilizado para criar aplicações Single Page. Além disso, ele permite ao programador criar suas próprias tags HTML.

As SPAs são aplicações completas, desenvolvidas em JavaScript, que funcionam quase como se estivessem sendo executadas nativamente no desktop. Não tem reload
